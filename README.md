# how-to-create-web-terminals

## How to run


1. Server
   1. cd ./server
   2. npm install (or) yarn install
   3. npm run start (or) yarn start
   
2. Client
   1. cd ./client
   2. npm install (or) yarn install
   3. npm run start (or) yarn start

### Important: Don't forget to change server address in [./client/src/index.js](./client/src/index.js)
